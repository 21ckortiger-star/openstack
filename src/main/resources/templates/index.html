About This Project

Hybrid Cloud AIaaS Dashboard는 프라이빗 클라우드(OpenStack)와 퍼블릭 클라우드(AWS)를 결합한 차세대 인프라 관리 솔루션입니다.

본 프로젝트는 IaC(Terraform)를 통한 인프라 자동 배포와 Ansible 기반의 구성 관리를 실현하며, 로컬의 자원 제약력을 극복하기 위해 AWS의 고지능형 AI 서비스(Lex, Rekognition)를 연동한 하이브리드 클라우드 모델을 제시합니다.
